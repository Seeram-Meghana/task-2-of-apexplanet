# Exploratory Data Analysis & Business Intelligence

## Objective
Performed Exploratory Data Analysis on sales dataset and created an interactive business dashboard using Excel.

## Tools Used
- Excel
- Pivot Tables
- Pivot Charts
- SQL Server

## KPIs
- Total Sales
- Total Orders
- Average Sales
- Total Quantity Sold

## Dashboard Features
- Sales by Region
- Product Category Analysis
- Payment Method Analysis
- Sales Channel Analysis

## Key Insights
- North region generated highest sales.
- Clothing category contributed highest revenue.
- Total sales exceeded ₹5 million.
